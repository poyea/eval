# <p align="center">eval</p>

<p align="center">
  Mathematical expression evaluator in Python.
</p>


## Usage
Read from a file:
```bash
$ cat text.txt
1**3+5
2**4**3
$ python main.py -f text.txt
6.0
15625.0
```
Command-line input:
```bash
$ python main.py 1 + 3**2 * 3
28.0
```
Interpreter:
```bash
$ python main.py
```
